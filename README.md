# MiniBankApplication
Jdbc based bank applications in which user can create account and login. After login he can perform further operations such as credit money ,debit money ,view balance , transfer money to other account etc.

Following classes are used
DbConnection:database connection using the DriverManager API.
Accounts: Model class to represent account entities.
BankManager: Contains the core Business Logic.
AccountUtility: A helper class for reusable tasks .
Main: The entry point.

Core Concepts Used
JDBC API
Exception Handling
